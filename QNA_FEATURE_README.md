# 问答社区功能说明

## 🎉 功能概述

这是一个功能完整、界面精美的问答社区系统，支持用户提问、回答、关注、点赞、采纳等完整的社区互动功能。

## ✨ 核心功能

### 1. 问题发布（我要提问）
- ✅ 支持选择资源类型（课程/网站/电子书）
- ✅ 支持关联资源ID
- ✅ 支持公开/付费专属问题
- ✅ 支持添加标签（最多5个）
- ✅ 推荐标签快速选择
- ✅ 字符计数（最多2000字）
- ✅ 表单验证（至少10个字符）
- ✅ 精美的UI设计和动画效果

### 2. 问题列表
- ✅ 多维度筛选（资源类型、资源ID、关键词）
- ✅ 分类Tab（全部/我的问题/我关注的/未回答/已回答）
- ✅ 问题卡片展示（标题、标签、统计数据）
- ✅ 付费问题特殊标识
- ✅ 已解决问题特殊样式
- ✅ 分页功能（支持切换每页数量）
- ✅ 相对时间显示（刚刚、X分钟前等）
- ✅ 响应式设计和悬停效果

### 3. 问题详情
- ✅ 完整的问题信息展示
- ✅ 关注/取消关注功能
- ✅ 回答列表展示
- ✅ 最佳回答特殊标识（绿色丝带）
- ✅ 点赞/取消点赞功能
- ✅ 采纳/取消采纳功能（仅问题作者）
- ✅ 提交回答功能
- ✅ 登录状态检测
- ✅ 固定底部回答输入框
- ✅ 字符计数和验证

## 🔌 API 集成

### 已实现的接口调用
1. **apiGetQnaTags** - 获取标签列表
2. **apiGetQuestionList** - 获取问题列表
3. **apiGetQuestionDetail** - 获取问题详情
4. **apiCreateQuestion** - 创建问题
5. **apiPublishQuestion** - 发布问题
6. **apiFollowQuestion** - 关注问题
7. **apiCancelFollowQuestion** - 取消关注
8. **apiSubmitAnswer** - 提交回答
9. **apiVoteAnswer** - 点赞回答
10. **apiCancelVoteAnswer** - 取消点赞
11. **apiSolveQuestion** - 采纳回答
12. **apiCancelSolve** - 取消采纳

### 接口规范
- ✅ 统一的错误处理
- ✅ 完整的请求参数验证
- ✅ 标准的响应格式处理
- ✅ Token自动携带
- ✅ 详细的错误提示

## 🚀 使用方式

### 访问问答社区
```

```

### 查看问题详情
```
/question_answer/detail/{questionId}
```

## 🔧 技术栈

- **框架**: Nuxt 3
- **UI库**: Naive UI
- **样式**: Scoped CSS
- **状态管理**: Vue 3 Composition API
- **路由**: Vue Router

## 📝 注意事项

1. **登录验证**: 部分功能需要登录后才能使用（关注、点赞、回答等）
2. **权限控制**: 采纳功能仅问题作者可见
3. **字符限制**: 
   - 问题内容：10-2000字符
   - 回答内容：10-2000字符
   - 标签：最多20字符，最多5个
4. **资源关联**: 选择资源类型后必须填写资源ID
