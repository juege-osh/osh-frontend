## 📝 使用说明

### 查看拼团活动
1. 点击导航栏"拼团"
2. 浏览拼团活动列表
3. 使用状态筛选器过滤活动
4. 点击活动卡片查看详情或参团

### 查看我的拼团
1. 点击导航栏"我的拼团"（需登录）
2. 查看我的拼团记录
3. 使用状态筛选器过滤记录
4. 点击记录查看详情

### 参与拼团
1. 在详情页或拼团列表找到拼团活动
2. 点击"立即参团"或"去拼团"
3. 确认参团信息
4. 完成支付（如需）

## ⚠️ 注意事项

### 后端接口依赖
确保以下后端接口已实现并可正常访问：
- GET /pc/group/activity/list
- GET /pc/group/activity/mylist
- GET /pc/group/work/detail

### 环境变量
检查 `composables/useHttp.js` 中的配置：
```javascript
baseURL: "http://demonuxtapi.dishait.cn/pc"
```
如需更换后端地址，请修改此配置。

- "我的拼团"页面需要登录
- 参与拼团需要登录并绑定手机号
